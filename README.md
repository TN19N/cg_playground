playground for CondinGame platform (https://www.codingame.com/)

## CodingGame environment:

### OS specifications: 
   * Linux 64bit multi-core architecture
   * max 100k characters of code size
   * max 768MB of RAM

### Language:

      - libc = "0.2.93"
      - rand = "0.8.3"
      - regex = "1.4.5"
      - time = "0.2.26"
